Chassis-Sass

Chassis is a Sass-based CSS framework for building fluid, responsive websites that are consistent in design throughout. "Who cares?" you ask. Well, Chassis has a few key differences from other CSS frameworks:

Typography is automatically optimized to your custom breakpoints

Chassis has good typography built-in. It automatically calculates optimal font-size and line-height for every viewport size, using its default set of breakpoints or your own custom set.

Every element of your web page adheres to these typography rules

Chassis is the only CSS framework in the world that bases the layout of all elements on typography rules. This means that not only will your typography automatically adjust to any screen size, so will every other element on the page.

Try it. Go ahead and shrink this window and watch how every element on the page scales appropriately. This all happens automatically, with no messy CSS or JavaScript.

Golden Ratio Typography

The typography rules inherent in Chassis weren't chosen arbitrarily; They are based on time-tested Golden Ratio Typography, optimized for the web. Not only that, but you can also specify different type scales if you choose.

In the event that you need more fine-grained control over individual pieces of typography, you can still specify font-sizes and line-heights as you normally would.

Get all the UI elements you need with Chassis Detailer

You can use Chassis's typography-based layout system by itself if you'd like, but if you need UI elements to work with, it is as simple as adding Chassis Detailer.

Chassis Detailer contains a suite of UI components, all of which work seamlessly with the Chassis typography-based layout system.

Simple, straightforward customization

All Detailer UI components are fully themable and can be extended with custom properties. It is easy to make many different versions of each component to suit your needs while maintaining automatic responsive behavior.

Only load the components you need

Chassis Detailer allows you to import exactly what you need, and nothing you don't, keeping CSS file size to a minimum.

The most semantic framework

Chassis applies the principles of SymbioCSS to provide the cleanest, most semantic HTML/CSS development experience possible. There are no cryptic class names to decipher, no proprietary HTML attributes, and every CSS selector reads like a short English phrase.

In addition, Chassis is totally self-contained. You are free to use any style of HTML or CSS in your own stylesheets without having to worry about scoping conflicts.

Table

Basic Golden Ratio Typography-compliant table component. You can turn the default border styling off and style table cells manually if desired. Ellipsis will work on all table cells.

You can optionally use the stack-below mixin to cause the table layout to stack below a specified breakpoint. For example: 

table {
  @include stack-below(small);
}

Flex Tiles

A fluid, responsive grid of equally-sized tiles with built-in numbering. 

<section class="tiles">
  <article></article>
  <article></article>
  <article></article>
  ...
</section>
.tiles {
  @include tiles(NUMBER_OF_COLUMNS);
  // additional styles

  @include min-viewport-width(medium) {
    @include tiles(DIFFERENT_NUMBER_OF_COLUMNS);
  }
}
